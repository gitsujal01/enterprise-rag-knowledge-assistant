# Enterprise RAG Knowledge Assistant

## 🚀 Overview
This project is a Retrieval-Augmented Generation (RAG) based assistant that helps users query documents using AI.

## 🛠️ Tech Stack
- Python
- FastAPI / Flask (your app)
- Vector DB
- Embeddings

## 📂 Features
- Document upload
- Chunking & embeddings
- Semantic search
- Auth system (login + roles)
